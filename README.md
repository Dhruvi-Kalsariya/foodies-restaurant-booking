# 🍽️ FOODIES - Restaurant Booking Website

FOODIES is a modern restaurant booking web application that allows users to explore restaurants, book tables, and manage reservations easily.

![Project Preview](screenshots/all-in-one.jpeg)

## 🚀 Features

* 🔍 Browse restaurants
* 📅 Table booking system
* 👤 User authentication (Login/Register)
* 📊 Admin dashboard
* 📱 Responsive design

---

## 🛠️ Tech Stack

* Frontend: HTML, CSS, JavaScript / React
* Backend: Node.js / ASP.NET
* Database: MySQL / SQL Server

---

## 📂 Project Structure

client/ → Frontend code
server/ → Backend API
database/ → SQL files

---

## ⚙️ Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/foodies-restaurant-booking.git
```

2. Install dependencies:

```bash
cd client
npm install

cd ../server
npm install
```

3. Run the project:

```bash
npm start
```
